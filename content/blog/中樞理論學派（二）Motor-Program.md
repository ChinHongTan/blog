現在來到了Motor Program的部份 大家看到Program這個詞很容易聯想到 程式 代碼之類\
沒錯 他這個就是指說在動作產生前 腦子已經想好一套流程 它就是Motor Program\
而因為這一套流程是先想好的 一樣\[不需要透過感覺回饋]{.red-3} 就能產生動作\
他這個Program包括了 肌肉收縮順序 產生的力量等等 \
而感覺回饋僅用來糾錯調整 不是用來刺激動作產生的\
而Program一但被執行 就必須被完成

以下將用動作程式代指Motor Program

那通用動作有三個理論 \
1\. 開鏈式系統 2. 通用動作程式 3. 階層特性

先來講開鏈式系統 像剛描述到的 一被執行則必須被完成 \
而他的特性就是 在動作結束前 都沒有回饋 達成後才有回饋\
對 這裡的特性就跟 <<第六課>> 文中的開放式回路相似\
而這個系統在穩定且可預測的環境中最有效

接下來到通用動作程式 顧名思義 就是他把動作拆分成各個參數 \
而有些是不可變的參數 裡面有1. 順序 2. 相對間隔/相對時間(比例) 3. 相對力量(比例)\
這三個不可變的參數構成通用動作程式\
假若有兩種動作的這三個不可變參數皆相同 那他們所使用的通用動作程式就是同一個\
即使有任一不可變參數不同 那兩者的通用動作程式也不一樣

舉一個例子 土跟士這兩個字 別看這兩字僅上下橫線長度不一樣\
但土字在寫字裡上橫比下橫用力 士則相反 下橫比上橫用力 \
即使順序(筆畫)跟相對間隔一樣 但兩者相對力量也不一樣\
所以是兩種通用動作程式

來突擊問一下 跑步跟走路兩種通用動作程式一樣嗎

而在通用動作程式裡可以改的參數

第一點 肌肉(部位) 意思就是 右撇子的你用左手寫同一個字 雖然字是比較醜 \
但同一個字的不變參數相同 不影響同一個通用動作程式在不同部位的應用

第二點 總力量 今天你拿毛筆寫字 你寫的總力度變大 但同一個字的相對力量比例不變 亦適用通用動作程式

第三點 總時間間隔 今天寫同一個字 把他寫大一點佔多點位置 但時長用更長\
而寫小一個字 佔的位置小 整體時間比例也會縮短

而最後講到階層特性 他跟前面周圍理論學派的階層理論不同\
這邊的階層是指說 今天我要做一個任務例如拿水杯 就會讓小的動作程式組合成拿水杯的任務\
動作程式如下

1)身體前彎的動作程式 2) 手臂往前伸的動作程式 3) 手指打開的動作程式 \
4) 抓握水杯的動作程式 5) 手臂回來的動作程式 \
這樣把一套動作所需的動作程式串聯在一起 就是階層特性

![1.00](https://raw.githubusercontent.com/ChinHongTan/blog/main/public/images/uploads/1774062985971-Screenshot_2026-03-21_111610.png)

臨床上 會注意不要打斷病人的動作 尤其是快速的動作\
且鼓勵病人自動的做出來(利用開鏈式)\
但當預到環境會變化的情境 則鼓勵利用閉鏈式(即<<第六課>>的閉鎖式回路 注重感覺回饋以調整糾錯\
假若病人因中風偏癱 動作程式沒有缺失情況下 可訓練病人用另一側肢體肌肉以執行動作程式\
可以在原本的動作程式中修改 不需要重新構築新的動作程式\
鼓勵善用階層特性
