**如何測試上下肢Brunnstrom stage 是否為Stage 2（第1/2期區分是有無聯合動作）**\
\
說話：主要是測有沒有聯合動作來區分一期或者第二期，聯合動作**上肢方向**相同，下肢除了[內收外展]{.underline}其他都相反\
那上肢要先從[Extensor]{.underline .red-3}測到再Flexor，讓好手用力抵抗阻力誘發壞手的動作\
下肢就先從[Flexor]{.underline .green-3}先測再到Extensor，一樣是用好腳用力抵抗阻力誘發壞腳動作

**測試上肢Brunnstrom stage是否為Stage 3的各項動作（紀錄角度）**\
\
說話：因為要先測弱的協同動作，以及從好邊先測，所以要先從[Extensor]{.underline}的協同動作開始測\
Extensor的動作的話：患者[同側手]{.underline}放在[同側大腿]{.underline}上，碰[對側]{.underline}膝蓋內側（[雙膝中間]{.underline}）並記錄各關節角度\
而Flexor的動作的話：患者同側手放在同側大腿上，往上碰同側[耳垂]{.underline}，並記錄各關節角度

**測試上肢Brunnstrom stage是否為Stage 4的各項動作**\
\
說話：\
第一個動作——手動到下背\
第二個動作——手向前平舉，掌心朝下\
第三個動作——Elbow Flex並且前臂做旋前和旋後

**測試上肢Brunnstrom stage是否為Stage 5的各項動作**\
\
說話：\
第一個動作——手側向平舉（掌心朝下）\
第二個動作——手側向平舉後（掌心朝下）-->前臂旋後（掌心朝上）\
第三個動作——手向前平舉再往前伸（Protraction）\
第四個動作——手向前高舉過頭

**測試下肢Brunnstrom stage是否為Stage 3的各項動作**\
\
說話：因為要先測弱的協同動作，以及從好邊先測，所以要先從[Flexor]{.underline}協同動作開始測\
Flexor的動作的話——請病人supine，並嘗試讓病人膝碰胸\
而Extensor的話——讓病人側躺，並用Hip Knee Flex的姿勢，做Knee Extend

**測試下肢Brunnstrom stage是否為Stage 4的各項動作**\
\
說話：\
第一個動作——在supine的姿勢下做hip abd\
第二個動作——坐姿下膝碰胸，那這個動作先測好邊再測壞邊\
第三個動作——坐姿下，腳踩在地上，向後滑，這個動作一樣先測好邊再測壞邊\
第四個動作——坐姿下，knee flex，做ankle dorsiflexion\
第五個動作——坐姿下，knee extend

**測試下肢Brunnstrom stage是否為Stage 5的各項動作**\
\
說話：\
第一個動作——坐姿下，要knee extend到腳跟離地下，做PF/DF\
第二個動作——坐姿下，僅腳跟碰地情況下，做eversion/inversion\
第三個動作——站姿下，一隻腳flex並且用後跟碰臀部

**Brunnstrom測試肩、肘、腕關節的被動動作感覺檢查**\
\
說話：那等下請你[閉上眼睛]{.underline}，然後我要動你的壞手，你的好手也要跟著做一樣的動作\
肩、肘的話——一手托住手肘，一手用兩指托住[掌心]{.underline}去移動\
腕的話——一手托住前臂，一手用二指抓[第二及第五掌骨]{.underline}去移動

**Brunnstrom測試手指Fingertip Recognition 及上肢之速度測試**\
\
說話：讓病人坐著，請病人[閉上眼睛]{.underline}，手放大腿上，用鉛筆的橡皮擦去碰病人的[壞側指尖]{.underline}，讓他說出是碰哪一根\
\
而速度測試的話：\
第一個——手放大腿靠近膝蓋的地方，然後去碰下巴再回到大腿，測五秒內能來回多少次\
第二個——手放大腿靠近髖部的地方，然後去碰對側膝蓋再回來，測五秒內能來回多少次

**Brunnstrom 測試足底 Sole sensation 檢查**\
\
說話：坐著，將壓舌板放在足下，測\
不放、放腳掌（橫）、放腳跟（橫）、放小趾側（竪）、放大腳趾側（竪）

**如何評估肘屈肌之肌肉痙攣（利用MAS）**\
\
說話：先將手放在extend，隨後慢慢的移動到Flex再慢慢回來，之後快速的將手拉動做Flex/extend三次感受阻力\
0分的話無阻力、1分的話只有末端有阻力、1+的話只有後半段有阻力\
2的話大半段都有阻力、3分的話很難拉動要用力才能拉開、4分的話完全拉不開

**請說明如何執行及判斷Functional Reach Test**\
\
說話：那首先病人要俱備這兩種能力，第一能不能獨自站立30秒以上，第二手能平舉起過90度\
接著就是身體直立，雙腳平行站立，握拳，手向前平舉90度，盡量讓病人向前伸，[測三次取平均值]{.underline}\
假如少於6inch的話，代表跌倒風險高\
假如6\~10inch的話，代表跌倒風險中
